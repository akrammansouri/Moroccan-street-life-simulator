# Morocco Street Hustle Simulator 🇲🇦

**From street rat to Moroccan tycoon — hustle, upgrade, dominate.**

## 🎮 Game Features
- 6+ Jobs (Food Seller, Taxi, Delivery, etc.)
- 10+ Vehicles (Scooter to Luxury Cars)
- 3 House Tiers (Medina Studio → Strip Penthouse)
- Daily Rewards & Lucky Spin
- Quest System
- PvP & Safe Zones
- Clan/Gang System
- Mobile + PC Optimized

## 🛠️ Setup Instructions
1. Download Roblox Studio
2. Create new Baseplate
3. Copy scripts from `/ServerScriptService` folder
4. Build map (or use template)
5. Publish to Roblox
